# Lennard-Jones Cluster Optimization

This directory contains two approaches for solving Lennard-Jones (LJ) cluster optimization problems:

## Folders

- **QA/** - Quantum Annealing approach for finding optimal LJ cluster configurations
- **RPQA/** - Replica-Pinned Quantum Annealing approach for enhanced convergence to global minima

## Reference Values

Reference structures and energies for Lennard-Jones clusters in LJ units can be found in the Cambridge Cluster Database:
https://doye.chem.ox.ac.uk/jon/structures/LJ.html

This database provides globally optimized structures for various cluster sizes, which can be used to validate the results obtained from both quantum annealing methods.
